# Graph ETA

Graph-Based ETA Optimization for Logistics Networks.

## Objective
Build a graph-based logistics intelligence system to:
- identify bottleneck hubs,
- improve ETA prediction,
- analyze corridor delays,
- engineer graph-based ML features.

## Tech Stack
- Python
- Pandas
- NetworkX
- Plotly
- Scikit-Learn
